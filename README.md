
# Document Management System (DMS)

## Project Description
This is a Document Management System built using the MEAN stack.
It allows users to upload, manage, and search documents with authentication and permissions.

## Features
- Upload documents (PDF, images)
- Add tags to documents
- Search documents
- User Authentication (Login/Register)
- Basic permissions
- Version tracking

## Tech Stack
- MongoDB
- Express.js
- Angular
- Node.js

## Setup Instructions

1. Download ZIP and extract

2. Backend Setup
cd backend
npm install
node server.js

3. Frontend Setup
cd dms-frontend
npm install
ng serve

4. Open in browser
http://localhost:4200

## Database
Make sure MongoDB is running

## Versions
Node.js v18+
Angular v15+
MongoDB v6+
